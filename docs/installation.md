# Installation

## Requirements

- Python `>=3.13,<3.14`

## Option 1: pip

```bash
pip install gitter-colony
```

## Option 2: uv

```bash
uv add gitter-colony
```

If you use `uv add`, run commands with `uv run ...` in that project.

## Verify

```bash
python -c "import gitter_py; print('gitter_py imported')"
```

Optional CLI check:

```bash
gitter --help
```
